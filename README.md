# Netflix Movie Analysis

## Description
This project analyzes Netflix movies data to gain insights into trends, popularity, ratings, and genres. Using Python and Jupyter Notebook, I processed and visualized the data to understand viewer preferences and movie performance metrics.

## Features
- **Genre-based insights** to identify trending categories  
- **Data visualizations** using bar plots, scatter plots, and heatmaps  
- Interactive exploration of Netflix movie dataset  

## Tools & Libraries
- **Python**  
- **Jupyter Notebook**  
- **Pandas** – Data manipulation  
- **Matplotlib & Seaborn** – Data visualization  
- **NumPy** – Numerical operations  

## Insights
- Popularity and rating trends over time  
- High-performing genres and patterns  
- Recommendations for movie watchers based on data
